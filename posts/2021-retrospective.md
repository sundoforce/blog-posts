# 0. 2021 회고를 시작하며

개발자로서 회고를 처음 해 봅니다.  
회사에서는 주니어에서 시니어로,
개인적으로는 컴퓨터과학 학사 학위를 취득 하면서 많은 격동의 시간을 보냈습니다.

음악커뮤니티 `https://muzia.net`을   public cloud 에서  
홈서버로 옮기면서 쿠버네티스를 설치하고
인프라에 대해서 얕게 경험도 해보았습니다.


# 1. 회사에서의 성장
## 1.1 주언어 변경
고객사의 요청에 따라 `Node.js` 를 사용 하다가 회사에서 많이 사용하는 언어를 잠시 사용하다가 대세인 Java를 익히려고 노력한 한 해 였습니다.

`Node.js` -> `Python`  -> `Java`로 바꾸면서  
Python로 개발 했던 것들을 `Spring Boot`로 porting 개발을 하였습니다.

* 공공포털데이터 연계
* Google Cloud Message - Push Service

## 1.2 빌드 및 배포 (CI/CD)
Java (Spring Web Project)를  `maven` 으로 변경하고    
`Jenkins`를 설치해서 docker 로 빌드하고 배포하는 작업을 하였습니다.

이전은 개발자들이 각자의 local 환경에서 jar 혹은 war 파일을   
was의 webapp 안에 직접 넣어야만 했습니다.

git에 개발 한것을 jenkins 버튼 한번으로   
devOps 서버에서 docker build 한것이  운영서버에서 docker run 되는 과정을 시도해 보면서 unix 와 shell에 조금 더 친숙해지는 계기가 되었습니다.

## 1.3 모바일 APP 개발
회사에서 누군가는 해야 하기에 도전했습니다.
`flutter` 의 WebView와 Firebase의 FCM 기능을 사용해 보았습니다.    
테스트하기 위해서  안드로이드 폰과  애플개발자 계정 1년을 개인 비용으로 지출 하였습니다.     
덕분에  개인 프로젝트인 `Muzia` APP을 제작 시도하게 됩니다.  
실패했지만 하이브리드로 시도 하지 않고, 시간과 여유가 된다면   
네이티브로 작성해 보고 싶습니다.


# 2. 지란지교 드림플랫폼 입상
하고 싶은 것과 일이 일치 한다면 그만큼 행복한 일은 없을 것입니다.  
저는 클래식 작곡과 출신 개발자이기 때문에 `음악과 IT의 융합`의  꿈이 있습니다.

재직중인 회사가 지란지교 패밀리가 되면서    
지란지교패밀리데이에 `드림플랫폼 - 코로나시대의 비대면 교육서비스` 영상을 제출해서 본선에 입상하였습니다.

코로나가 시작되면서 아내가 중동 `아부다비` 와 `대전시`에서 지금까지 매주 1회 바이올린 레슨을 하고 있습니다.

이 과정을 담아 보았고, 코로나시대 이전 부터 꿈 꿔왔던 플랫폼을 구상했습니다.

# 6. 백엔드 개발에서 클라우드로 전환

프로그램들이 잘 돌아갈 수 있게 시스템설계를 잘 해보겠습니다.   
많은 기대가 되고, 팀분위기가 저랑 잘 맞을 것 같습니다.

2021년에 SI를 벗어나고자 여러가지 일을 했습니다.
> 아무일도 하지 않으면, 아무일도 일어나지 않는다.

## 6.1 코딩테스트
카카오, 쿠팡, 세트렉아이 등 규모있는 회사에 코딩테스트를 보았습니다.

코딩테스트를 감을 익힌 한해 였습니다.
> 나는 실패하지 않는다, 배움을 얻는다  
성공하거나 배우거나 인데, 저는 배움을 얻었습니다.

카카오 코딩테스트는  해커랭크에서 `python`으로  
쿠팡은 프로그래머스에서 `Python` 으로 1번, 언어를 `Java` 변경해서 총 2번 보았습니다.

## 6.2 유튜브 크리에이터
